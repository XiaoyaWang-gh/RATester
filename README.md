# Enhancing LLMвЂ™s Ability to Generate More Repository-Aware Unit Tests Through Global Contextual Information Injection

This repo contains both the unit tests generated by our study along with the code used to run the experiment.

## Prerequisites

### Install Go 1.22.4 and gopls

```
wget https://golang.org/dl/go1.22.4.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.22.4.linux-amd64.tar.gz
vim ~/.bashrc
export PATH=$PATH:/usr/local/go/bin
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
source ~/.bashrc
go install golang.org/x/tools/gopls@latest
```

### Download projects

```
Download Golang projects from https://figshare.com/s/7f402019c7a8a6e15912
```

## Run the gopls
Configure the gopls address in ./jsonrpc/provider_settings.json.

```
cd jsonrpc
go run gopls.go --listen 127.0.0.1:2000
```

## Generate the unit tests
Example usage to run RATester:
model: Model to use for unit test generation, should be one of [CodeLlama, DeepSeek-Coder, Magicoder].

```
python main.py --model DeepSeek-Coder --gopls-endpoint http://127.0.0.1:2000 --results-root ./results
```

## Execute the unit test
model: Model to use for unit test generation, should be one of [CodeLlama, DeepSeek-Coder, Magicoder].
project: Project to be tested. should be one of [beego, echo, fiber, frp, gin, hugo, nps, traefik].

```
model=DeepSeek-Coder
project=beego
sh run_test.sh $model $project  
```
